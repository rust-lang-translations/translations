[expr.array.constructor]

_배열 표현식_은 배열을 생성합니다. 배열 표현식에는 두 가지 형식이 있습니다.

[expr.array.array]

첫 번째 형식은 배열의 모든 값을 나열합니다.

[expr.array.array-syntax]

이 형식의 구문은 대괄호 안에 쉼표로 구분된 동일한 타입의 표현식 목록입니다.

[expr.array.array-behavior]

이것은 각 값을 작성된 순서대로 포함하는 배열을 생성합니다.

[expr.array.repeat]

두 번째 형식의 구문은 대괄호 안에 세미콜론(;)으로 구분된 두 개의 표현식입니다.

[expr.array.repeat-operand]

; 앞의 표현식을 _반복 피연산자_라고 합니다.

[expr.array.length-operand]

; 뒤의 표현식을 _길이 피연산자_라고 합니다.

[expr.array.length-restriction]

이것은 usize 타입이어야 하며, 리터럴이나 상수 아이템과 같은 상수 표현식이어야 합니다.

[expr.array.repeat-behavior]

이 형식의 배열 표현식은 길이 피연산자의 값만큼의 길이를 가지며, 각 요소가 반복 피연산자의 복사본인 배열을 생성합니다. 즉, [a; b]는 a 값의 복사본 b개를 포함하는 배열을 생성합니다.

[expr.array.repeat-copy]

길이 피연산자의 값이 1보다 큰 경우, 반복 피연산자의 타입이 Copy이거나 상수 아이템에 대한 경로여야 합니다.

[expr.array.repeat-const-item]

반복 피연산자가 상수 아이템인 경우, 길이 피연산자의 값만큼 평가됩니다.

[expr.array.repeat-evaluation-zero]

그 값이 0이면, 상수 아이템은 전혀 평가되지 않습니다.

[expr.array.repeat-non-const]

상수 아이템이 아닌 표현식의 경우, 정확히 한 번 평가되고 결과는 길이 피연산자의 값만큼 복사됩니다.

#![allow(unused)]
fn main() {
[1, 2, 3, 4];
["a", "b", "c", "d"];
[0; 128];              // 128개의 0을 가진 배열
[0u8, 0u8, 0u8, 0u8,];
[[1, 0, 0], [0, 1, 0], [0, 0, 1]]; // 2차원 배열
const EMPTY: Vec<i32> = Vec::new();
[EMPTY; 2];
}
[expr.array.index]

배열 및 슬라이스 인덱싱 표현식

구문
인덱스표현식 :
   표현식 [ 표현식 ]

[expr.array.index.array]

배열 및 슬라이스 타입의 값은 그 뒤에 usize 타입의 대괄호로 묶인 표현식(인덱스)을 작성하여 인덱싱할 수 있습니다. 배열이 가변인 경우 결과 메모리 위치에 할당할 수 있습니다.

[expr.array.index.trait]

다른 타입의 경우 인덱스 표현식 a[b]는 *std::ops::Index::index(&a, b)와 동일하며, 가변 장소 표현식 컨텍스트에서는 *std::ops::IndexMut::index_mut(&mut a, b)와 동일합니다. 메서드와 마찬가지로, Rust는 구현을 찾기 위해 a에 대해 반복적으로 역참조 작업을 수행합니다.

[expr.array.index.zero-index]

배열과 슬라이스의 인덱스는 0부터 시작합니다.

[expr.array.index.const]

배열 접근은 상수 표현식이므로 상수 인덱스 값을 사용하여 컴파일 타임에 범위를 확인할 수 있습니다. 그렇지 않으면 런타임에 확인이 수행되며, 실패할 경우 스레드가 _패닉 상태_가 됩니다.

#![allow(unused)]
fn main() {
// 린트는 기본적으로 거부(deny)됩니다.
#![warn(unconditional_panic)]

([1, 2, 3, 4])[2];        // 3으로 평가됩니다

let b = [[1, 0, 0], [0, 1, 0], [0, 0, 1]];
b[1][2];                  // 다차원 배열 인덱싱

let x = (["a", "b"])[10]; // 경고: 인덱스가 범위를 벗어남

let n = 10;
let y = (["a", "b"])[n];  // 패닉 발생

let arr = ["a", "b"];
arr[10];                  // 경고: 인덱스가 범위를 벗어남
}
[expr.array.index.trait-impl]

배열 인덱스 표현식은 Index 및 IndexMut 트레잇을 구현하여 배열 및 슬라이스 이외의 타입에 대해 구현할 수 있습니다.
