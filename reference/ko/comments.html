[comments.normal]

비 문서 주석

주석은 일반적인 C++ 스타일의 라인 (//) 및 블록 (/* ... */) 주석 형식을 따릅니다. 중첩된 블록 주석이 지원됩니다.

[comments.normal.tokenization]

비 문서 주석은 공백의 한 형태로 해석됩니다.

[comments.doc]

문서 주석

[comments.doc.syntax]

정확히 세 개의 슬래시(///)로 시작하는 라인 문서 주석과 블록 문서 주석(/** ... */), 이 두 가지 외부 문서 주석은 doc 속성을 위한 특별한 구문으로 해석됩니다.

[comments.doc.attributes]

즉, 이들은 주석 본문 주위에 #[doc="..."]를 작성하는 것과 동일합니다. 즉, /// Foo는 #[doc="Foo"]로, /** Bar */는 #[doc="Bar"]로 바뀝니다. 따라서 외부 속성을 허용하는 것 앞에 나타나야 합니다.

[comments.doc.inner-syntax]

//!로 시작하는 라인 주석과 /*! ... */ 블록 주석은 뒤따르는 항목이 아닌 주석의 부모에 적용되는 문서 주석입니다.

[comments.doc.inner-attributes]

즉, 이들은 주석 본문 주위에 #![doc="..."]를 작성하는 것과 동일합니다. //! 주석은 일반적으로 소스 파일을 차지하는 모듈을 문서화하는 데 사용됩니다.

[comments.doc.bare-crs]

문서 주석에는 U+000D (CR) 문자가 허용되지 않습니다.

참고: rustdoc에서 예상하는 바와 같이 문서 주석에 마크다운을 포함하는 것이 일반적입니다. 그러나 주석 구문은 내부 마크다운을 존중하지 않습니다. /** `glob = "*/*.rs";` */는 첫 번째 */에서 주석을 종료하며, 나머지 코드는 구문 오류를 발생시킵니다. 이는 라인 문서 주석에 비해 블록 문서 주석의 내용을 약간 제한합니다.

참고: U+000D (CR) 문자와 그 뒤에 바로 오는 U+000A (LF) 문자의 시퀀스는 이전에 단일 U+000A (LF)로 변환되었을 것입니다.

예시

#![allow(unused)]
fn main() {
//! 이 크레이트의 암시적 익명 모듈에 적용되는 문서 주석

pub mod outer_module {

    //!  - 내부 라인 문서
    //!! - 여전히 내부 라인 문서 (하지만 시작에 느낌표가 있음)

    /*!  - 내부 블록 문서 */
    /*!! - 여전히 내부 블록 문서 (하지만 시작에 느낌표가 있음) */

    //   - 단순 주석
    ///  - 외부 라인 문서 (정확히 슬래시 3개)
    //// - 단순 주석

    /*   - 단순 주석 */
    /**  - 외부 블록 문서 (정확히) 별표 2개 */
    /*** - 단순 주석 */

    pub mod inner_module {}

    pub mod nested_comments {
        /* Rust에서는 /* /* 주석을 중첩할 수 있습니다 */ */ */

        // 세 가지 유형의 블록 주석은 다른 유형 내에 포함되거나 중첩될 수 있습니다:

        /*   /* */  /** */  /*! */  */
        /*!  /* */  /** */  /*! */  */
        /**  /* */  /** */  /*! */  */
        pub mod dummy_item {}
    }

    pub mod degenerate_cases {
        // 비어 있는 내부 라인 문서
        //!

        // 비어 있는 내부 블록 문서
        /*!*/

        // 비어 있는 라인 주석
        //

        // 비어 있는 외부 라인 문서
        ///

        // 비어 있는 블록 주석
        /**/

        pub mod dummy_item {}

        // 비어 있는 2-별표 블록은 문서 블록이 아니라 블록 주석입니다.
        /***/

    }

    /* 다음은 외부 문서 주석이 문서를 받을 항목을 요구하기 때문에 허용되지 않습니다 */

    /// 내 항목은 어디에 있나요?
  mod boo {}
}
}
