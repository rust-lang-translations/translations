The Rust Reference

[type.bool]

불리언 타입

#![allow(unused)]
fn main() {
let b: bool = true;
}
[type.bool.intro]

불리언 타입 또는 _bool_은 _true_와 _false_라고 불리는 두 가지 값 중 하나를 가질 수 있는 기본 데이터 타입입니다.

[type.bool.literal]

이 타입의 값은 같은 이름의 값에 해당하는 true 및 false 키워드를 사용한 리터럴 표현식을 통해 생성될 수 있습니다.

[type.bool.namespace]

이 타입은 bool이라는 이름으로 언어 프렐류드의 일부입니다.

[type.bool.layout]

불리언 타입을 가진 객체는 크기와 정렬이 각각 1입니다.

[type.bool.repr]

false 값은 비트 패턴 0x00을 가지며, true 값은 비트 패턴 0x01을 가집니다. 불리언 타입을 가진 객체가 그 외의 비트 패턴을 갖는 것은 정의되지 않은 동작(undefined behavior)입니다.

[type.bool.usage]

불리언 타입은 다양한 표현식에서 많은 피연산자의 타입으로 사용됩니다:

[type.bool.usage-condition]
  • if 표현식 및 while 표현식의 조건 피연산자
[type.bool.usage-lazy-operator]
  • 지연 불리언 연산자 표현식의 피연산자

참고: 불리언 타입은 열거형 타입과 유사하게 작동하지만 열거형은 아닙니다. 실제로 이는 주로 생성자가 타입에 연관되지 않음을 의미합니다 (예: bool::true).

[type.bool.traits]

모든 기본 타입과 마찬가지로, 불리언 타입은 Clone, Copy, Sized, Send, Sync 트레잇을 구현합니다.

참고: 라이브러리 연산에 대해서는 표준 라이브러리 문서를 참조하십시오.

[type.bool.validity]

비트 유효성

bool의 단일 바이트는 초기화됨이 보장됩니다 (다시 말해, transmute::<bool, u8>(...)는 항상 안전(sound)합니다. 하지만 일부 비트 패턴은 유효하지 않은 bool 값이므로, 그 역은 항상 안전하지는 않습니다).
